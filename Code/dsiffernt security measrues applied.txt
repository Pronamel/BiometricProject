

Current Security Measures


1. CORS Protection (lines 38-55)
Restricts which domains can call the API
Development vs Production policies
Prevents unauthorized website access

2. JWT Authentication & Authorization (lines 18-35)
JWT tokens for session management
Role-based access ("voter", "official")
Token validation with secret key
Issuer/Audience validation
Token expiration enforcement

3. Role-Based Access Control
Voter endpoints require voter role (line 278)
Official endpoints require official role (lines 296, 312)
Different token expiration times:
Officials: 24 hours
Voters: 4 hours

4. Request Logging (lines 85-95)
Logs all API requests with timestamps
Tracks client IP addresses
Method and path logging
5. One-Time Access Codes (line 281)
Access codes are removed after single use
Prevents code reuse attacks
